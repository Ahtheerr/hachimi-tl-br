## 📖 Como Instalar (Tutorial)

Siga os passos abaixo para aplicar a tradução no seu jogo:

1. **Baixe o Hachimi Edge:** Faça o download da versão mais recente [aqui](https://github.com/kairusds/Hachimi-Edge/releases).
2. **Instale a Ferramenta:** Você pode instalar o Hachimi utilizando o instalador automático fornecido ou realizar a instalação manualmente.
4. **Baixe a Tradução:** Siga os prompts do Hachimi para fazer o download da tradução em Português.
5. **Seja Feliz! 🎉** Aproveite o jogo!

## 🛠️ Solução de Problemas (Bugs Conhecidos)

### O jogo crasha ao abrir (Conflito de GUI + Tela Cheia)

Se o seu jogo fechar sozinho logo ao iniciar, isso geralmente é causado por um conflito entre a interface (GUI) do Hachimi e o modo de tela cheia do jogo.

**Como resolver:**

* Saia do modo tela cheia (use `Alt + Enter` ou mude nas configurações do jogo antes de instalar o mod).
* Se preferir desativar a interface do Hachimi para evitar o crash permanentemente:
1. Abra o jogo uma vez para que o arquivo de configuração seja gerado.
2. Vá até a pasta `hachimi` e abra o arquivo `Config.json`.
3. Procure pela opção da GUI e mude o valor para `false` (ou use o menu de configurações do próprio Hachimi dentro do jogo, se conseguir acessar antes do crash).
